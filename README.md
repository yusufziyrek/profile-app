# Profile App

## Overview
- React Native profile card showcase built with Expo.
- Displays three historical figures with tap alerts.
- Implements reusable components and SafeAreaView layout.

## Project Structure
- `App.js`: Entry point rendering the scrollable list of profile cards.
- `components/ProfileCard.js`: Reusable card component for profile data.
- `assets/`: Local images used by the cards.

## Getting Started
1. Install dependencies: `npm install` or `npx expo install` as required.
2. Start the development server: `npx expo start`.
3. Open the Expo Go app and scan the QR code to view the project on a device.
